📝 Week 7 Summary: Passive Voice

Passiv und Passiversatzformen - Quick Reference

🎯 Key Concept

Passive Voice shifts focus from who does it to what happens.

Active: Daniel liest das Buch. (Daniel reads the book.)

Passive: Das Buch wird gelesen. (The book is being read.)

📋 All Passive Tenses at a Glance

Tense Formation Example
Präsens werden + Partizip II Das Buch wird gelesen.
Präteritum wurden + Partizip II Das Buch wurde gelesen.
Perfekt sein + Partizip II + worden Das Buch ist gelesen worden.
Plusquamperfekt waren + Partizip II + worden Das Buch war gelesen worden.
Modal (Präs.) Modal + Partizip II + werden Das Buch muss gelesen werden.
Modal (Prät.) Modal (past) + Partizip II + werden Das Buch musste gelesen werden.
Futur I werden + Partizip II + werden Das Buch wird gelesen werden.
Futur II werden + Partizip II + worden + sein Das Buch wird gelesen worden sein.

👤 Adding the Agent: von vs. durch

von + Dative

Use for people/active agents

  • Das Buch wird von Daniel gelesen.
  • Das Haus wird von der Firma gebaut.
  • Die Regel wird vom Lehrer erklärt.

durch + Accusative

Use for means/causes

  • Das Fenster wurde durch einen Sturm zerbrochen.
  • Die Stadt wurde durch ein Erdbeben zerstört.
  • Das Problem wird durch Training gelöst.

💡 Golden Rules & Memory Tricks

🎯 Rule 1: "worden" not "geworden"

In passive perfect, use worden (without ge-)

✅ ist gelesen worden

❌ ist gelesen geworden

🎯 Rule 2: Always "sein" in perfect

Passive perfect uses sein, never haben

✅ Das Buch ist gelesen worden.

❌ Das Buch hat gelesen worden.

🎯 Rule 3: Don't forget "werden"

Modal verbs need werden at the end

✅ muss gelesen werden

❌ muss gelesen

🎯 Rule 4: Double "werden" is normal

Futur I passive has werden twice

✅ wird gelesen werden

will be read

⚡ Quick Examples by Tense

Präsens (Present)

  • Das Auto wird repariert. (The car is being repaired.)
  • Die Tür wird geschlossen. (The door is being closed.)
  • Das Essen wird gekocht. (The food is being cooked.)

Präteritum (Past)

  • Der Dieb wurde gefunden. (The thief was found.)
  • Das Auto wurde verkauft. (The car was sold.)
  • Der Patient wurde behandelt. (The patient was treated.)

Perfekt

  • Die Tür ist geschlossen worden. (The door has been closed.)
  • Das Problem ist gelöst worden. (The problem has been solved.)
  • Das Auto ist repariert worden. (The car has been repaired.)

Modal Verbs

  • Das Buch muss gelesen werden. (The book must be read.)
  • Die Tür kann geöffnet werden. (The door can be opened.)
  • Das Formular soll ausgefüllt werden. (The form should be filled out.)

⚠️ Top 4 Mistakes to Avoid

❌ Mistake 1

Wrong: ist gelesen geworden

Right: ist gelesen worden

❌ Mistake 2

Wrong: muss gelesen

Right: muss gelesen werden

❌ Mistake 3

Wrong: Das Buch hat gelesen worden

Right: Das Buch ist gelesen worden

❌ Mistake 4

Wrong: wurde von einem Sturm zerstört

Right: wurde durch einen Sturm zerstört

🔄 Active → Passive Transformation

Steps:

  1. Move accusative object to subject position
  2. Add correct form of werden
  3. Change main verb to Partizip II
  4. Optionally add agent with von/durch

Examples:

Active Passive
Ich lese das Buch. Das Buch wird gelesen.
Der Lehrer erklärt die Regel. Die Regel wird erklärt.
Die Firma baut ein Haus. Ein Haus wird gebaut.

✏️ Quick Practice

Transform to Passive:

  1. Der Mechaniker repariert das Auto. → Das Auto wird repariert.
  2. Die Polizei fand den Dieb. → Der Dieb wurde gefunden.
  3. Ich habe die Tür geschlossen. → Die Tür ist geschlossen worden.
  4. Man muss die Rechnung bezahlen. → Die Rechnung muss bezahlt werden.
