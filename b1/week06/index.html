📎 Week 06: Konsekutivsätze

Consecutive Clauses

Master German consecutive clauses to express results and consequences: show how one action leads to another

🎯 Learning Objectives

  • Understand consecutive clauses (Konsekutivsätze)
  • Learn key conjunctions: so dass, sodass, so...dass
  • Master the difference between purpose and result clauses
  • Practice expressing consequences and results
  • Apply consecutive clauses in complex sentences

📚 Was sind Konsekutivsätze?

What are Consecutive Clauses?

Definition

Konsekutivsätze sind Nebensätze, die eine Folge oder ein Ergebnis ausdrücken. Sie antworten auf die Frage "Mit welcher Folge?" oder "Mit welchem Ergebnis?" (With what consequence? / With what result?).

English: Consecutive clauses are subordinate clauses that express a consequence or result. They answer the question "With what consequence?" or "With what result?"

Basic Structure

Pattern: Hauptsatz + so dass/sodass + Nebensatz (Verb am Ende)

English Pattern: Main clause + so that + subordinate clause (verb at the end)


Example:

Es regnet stark, so dass wir zu Hause bleiben.

English: It's raining heavily, so that we stay at home.

🔗 Key Conjunctions for Consecutive Clauses

1. so dass

English: so that

The most common way to express consequence

Usage: Written as two separate words
Meaning: Expresses a result or consequence

2. sodass

English: so that

Alternative spelling, more modern

Usage: Written as one word
Meaning: Same as "so dass" - result/consequence

3. so...dass

English: so...that

Used with adjectives/adverbs to emphasize degree

Pattern: so + adjective/adverb + dass
Meaning: To such a degree that...

📐 1. So dass / Sodass (Basic Consecutive Clauses)

Usage Rules

  • "so dass" and "sodass" have identical meaning
  • Both express consequence or result
  • The subordinate clause shows what happens as a result
  • Verb goes to the end in the consecutive clause

Examples with "so dass"

🎯 Everyday Situations

German: Ich bin krank, so dass ich nicht arbeiten kann.

English: I am sick, so that I cannot work.


German: Es ist sehr kalt, so dass wir den Mantel anziehen.

English: It is very cold, so that we put on coats.


German: Der Film war langweilig, so dass viele Leute das Kino verlassen haben.

English: The movie was boring, so that many people left the cinema.

🎯 Work and Study

German: Er lernt fleißig, so dass er die Prüfung besteht.

English: He studies diligently, so that he passes the exam.


German: Sie spricht langsam, so dass alle sie verstehen.

English: She speaks slowly, so that everyone understands her.


German: Das Projekt ist kompliziert, so dass wir mehr Zeit brauchen.

English: The project is complicated, so that we need more time.

Examples with "sodass" (One Word)

German: Der Zug hatte Verspätung, sodass ich zu spät kam.

English: The train was delayed, so that I arrived late.


German: Sie macht den Fernseher leiser, sodass das Baby schläft.

English: She turns down the TV, so that the baby sleeps.


German: Wir fahren mit dem Auto, sodass wir schneller ankommen.

English: We drive by car, so that we arrive faster.


German: Das Restaurant ist geschlossen, sodass wir woanders essen müssen.

English: The restaurant is closed, so that we have to eat somewhere else.

⚡ 2. So...dass (Intensified Consecutive Clauses)

So...dass Structure

Pattern: Main clause + so + adjective/adverb + dass + consequence

Usage: Emphasizes the degree or extent that leads to a consequence

English equivalent: "so...that" / "such...that"

Examples with "so...dass"

🎯 With Adjectives

German: Das Buch ist so interessant, dass ich es in einer Nacht gelesen habe.

English: The book is so interesting that I read it in one night.


German: Der Kaffee ist so heiß, dass ich ihn nicht trinken kann.

English: The coffee is so hot that I cannot drink it.


German: Sie ist so müde, dass sie sofort einschläft.

English: She is so tired that she falls asleep immediately.

🎯 With Adverbs

German: Er spricht so schnell, dass niemand ihn versteht.

English: He speaks so fast that no one understands him.


German: Es regnet so stark, dass wir nass werden.

English: It's raining so hard that we get wet.


German: Sie arbeitet so viel, dass sie keine Freizeit hat.

English: She works so much that she has no free time.

Special Cases with "so...dass"

German: Der Koffer ist so schwer, dass ich ihn nicht tragen kann.

English: The suitcase is so heavy that I cannot carry it.


German: Das Konzert war so laut, dass meine Ohren wehgetan haben.

English: The concert was so loud that my ears hurt.


German: Die Straßen sind so glatt, dass viele Unfälle passieren.

English: The roads are so slippery that many accidents happen.

🆚 Consecutive Clauses vs. Purpose Clauses

Important Distinction: Result vs. Intention

⚠️ Purpose Clauses (Finalsätze)

Conjunctions: damit, um...zu

Meaning: Express intention or purpose

Question: Wozu? Zu welchem Zweck? (For what purpose?)


Example:

Ich lerne Deutsch, damit ich in Deutschland arbeiten kann.

I learn German so that I can work in Germany.

→ INTENTION/GOAL

✅ Consecutive Clauses (Konsekutivsätze)

Conjunctions: so dass, sodass, so...dass

Meaning: Express result or consequence

Question: Mit welcher Folge? (With what result?)


Example:

Ich lerne viel Deutsch, so dass ich jetzt gut sprechen kann.

I learn a lot of German, so that I can now speak well.

→ RESULT/CONSEQUENCE

Direct Comparison Examples

🎯 Purpose (damit)

German: Sie kauft ein Auto, damit sie schneller zur Arbeit kommt.

English: She buys a car so that she can get to work faster.

(Goal/Intention)


German: Ich spreche laut, damit alle mich hören.

English: I speak loudly so that everyone can hear me.

(Goal/Intention)

🎯 Result (so dass)

German: Sie hat ein Auto gekauft, so dass sie jetzt schneller zur Arbeit kommt.

English: She bought a car, so that she now gets to work faster.

(Result/Consequence)


German: Ich spreche zu laut, so dass alle mich hören.

English: I speak too loudly, so that everyone hears me.

(Result/Consequence)

📊 Overview Table: Consecutive Conjunctions

Conjunction English Usage Word Order Example
so dass so that Basic result/consequence Verb at end Es regnet, so dass wir drinnen bleiben.
sodass so that Same as "so dass" (modern spelling) Verb at end Der Zug kommt, sodass wir gehen können.
so...dass so...that Emphasizes degree/extent Verb at end Er ist so müde, dass er schläft.

⚠️ Common Mistakes & Tips

❌ Häufige Fehler (Common Mistakes)

  • Confusing purpose and result:
    ❌ Ich arbeite hart, damit ich müde bin. (wrong - this is a result, not purpose)
    ✅ Ich arbeite hart, so dass ich müde bin.
    ✅ Ich arbeite hart, damit ich Erfolg habe. (purpose)
    English: I work hard, so that I am tired / I work hard so that I have success.
  • Wrong word order:
    ❌ Es ist kalt, so dass ich ziehe einen Mantel an.
    ✅ Es ist kalt, so dass ich einen Mantel anziehe.
    English: It's cold, so that I put on a coat.
  • Missing "so" in "so...dass":
    ❌ Der Film ist interessant, dass ich ihn zweimal anschaue.
    ✅ Der Film ist so interessant, dass ich ihn zweimal anschaue.
    English: The movie is so interesting that I watch it twice.

✅ Memory Tips

  • Result vs. Purpose test: Ask "Was passierte dann?" (What happened then?) → Consecutive clause
  • "So dass" = "so that" literally: Both express result/consequence
  • "So...dass" pattern: Think "so tired that...", "so loud that..."
  • Verb position: Always at the end in subordinate clauses!
  • Time logic: Consecutive clauses describe what actually happened (result)

🎯 Practice Exercises

Exercise 1: Choose the correct conjunction

1. Es ist sehr heiß, _____ wir ins Schwimmbad gehen. (so dass/damit)

2. Sie spricht _____ leise, _____ ich sie nicht verstehe. (so...dass/damit...dass)

3. Ich kaufe Blumen, _____ meine Frau sich freut. (so dass/damit)

4. Der Film war langweilig, _____ viele Leute gegangen sind. (sodass/damit)

5. Er läuft schnell, _____ er den Bus noch erreicht. (so dass/damit)

✅ Answers

1. so dass - It's very hot, so that we go to the swimming pool. (result)

2. so...dass - She speaks so quietly that I don't understand her. (degree + result)

3. damit - I buy flowers so that my wife is happy. (purpose)

4. sodass - The movie was boring, so that many people left. (result)

5. damit - He runs fast so that he still catches the bus. (purpose)

Exercise 2: Complete the sentences

1. Das Wetter ist so schlecht, dass _____ (wir/zu Hause/bleiben)

2. Sie arbeitet viel, so dass _____ (sie/wenig Zeit/haben)

3. Der Koffer ist so schwer, dass _____ (ich/ihn/nicht/tragen/können)

4. Es regnet stark, sodass _____ (alle Leute/schnell/laufen)

✅ Answers

1. wir zu Hause bleiben - The weather is so bad that we stay at home.

2. sie wenig Zeit hat - She works a lot, so that she has little time.

3. ich ihn nicht tragen kann - The suitcase is so heavy that I cannot carry it.

4. alle Leute schnell laufen - It's raining hard, so that all people run quickly.
